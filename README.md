# 网易图床
网易严选图床，压缩率很高，且会修改图片属性，上传大小限制在8M以内

### 源码及发行版
1. 已内置解决跨域问题
2. 内部捕获了异常响应到json
3. 全异步构建

### Windows平台运行
双击wy_cloud.exe运行即可

### Linux运行
1. 如有可执行权限，此处省略
``` shell
chmod +x wy_cloud.bin
```
2. 运行
``` shell
./wy_cloud.bin
```
